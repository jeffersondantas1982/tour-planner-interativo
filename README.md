## Imersão Dev com Google Gemini 10ª Edição"
## Introdução da Funcionalidade "Monte Seu Passeio"

Este Projeto adiciona a funcionalidade central para montar roteiros de viagem de forma dinâmica e interativa, conforme o desafio da Imersão Dev com Google Gemini. O principal objetivo é proporcionar uma experiência de usuário (UX) moderna e intuitiva.

### Destaques da Implementação:
* **Separação de Preocupações (CSS/HTML):** O código CSS foi extraído para um arquivo dedicado (`styles.css`), garantindo maior manutenibilidade, seguindo as melhores práticas de desenvolvimento.
* **Interatividade e Seleção:** Implementação de cards clicáveis para a seleção de passeios/transfers, com feedback visual claro (`.card.selected`).
* **Experiência Flutuante:** Adição de um **Resumo do Carrinho Flutuante** (`.floating-cart-summary`) que oferece visibilidade constante do orçamento estimado.
* **Busca Intuitiva:** Inclusão de um campo de pesquisa para filtrar rapidamente entre Passeios Coletivos, Privativos e Transfers.
* **Responsividade Aperfeiçoada:** Ajustes finos no CSS para garantir que a interface seja 100% responsiva em todos os dispositivos, do mobile ao desktop.
* **Destaque do Desenvolvedor:** Manutenção do bloco de destaque dinâmico para os avaliadores, seguindo as diretrizes do concurso.

### Próximos Passos:

* Popular as seções de cards com os dados reais dos passeios.
